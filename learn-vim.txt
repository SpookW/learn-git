1.launch vim 
type in command line: $   vim
2.open filename and create the file with name:filename
type in command line: $  vim filename
3.file orders
----1-----open one file
type in command line: $  vim filename
----2-----open many file
type in command line: $  vim filename1 filename2 ...
4.vim pattern
----1----normal pattern
enter <esc> or <ctrl+[>
----2---insert pattern
enter <i>
----3---visual pattern
enter <v> or <ctrl+v>

5.exit vim
in normal pattern 
:wq    ||save and exit
:q!    ||force exit and ignore every changes
:e!    ||give up every changes and open the origin file.
