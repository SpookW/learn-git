1.launch vim 
type in command line: $   vim
2.open filename and create the file with name:filename
type in command line: $  vim filename
3.file orders
----1-----open one file
type in command line: $  vim filename
----2-----open many file
type in command line: $  vim filename1 filename2 ...
4.vim pattern
----1----normal pattern
enter <esc> or <ctrl+[>
----2---insert pattern
enter <i>
----3---visual pattern
enter <v> or <ctrl+v>
5.exit vim
in normal pattern 
:wq    ||save and exit
:q!    ||force exit and ignore every changes
:e!    ||give up every changes and open the origin file.
6.move orders
in normal pattern 
$      ||move to the end of this line
0      ||nunmber0  move to the start of this line
G or shift+g  || move to the begin of last line of this page
7.undo order
in normal pattern
u      ||undo last order
